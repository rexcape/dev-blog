最近重装了一遍系统，记录一下每次重装系统反复配置的软件，本文记录如何用 oh-my-posh 美化 PowerShell

开放许可

首次在计算机上启动 Windows PowerShell 时，现用执行策略很可能是 Restricted（默认设置），这个策略是禁止运行任何脚本的，要想打开，只需在有管理员权限的 PowerShell 下输入 set-executionpolicy remotesigned即可。
