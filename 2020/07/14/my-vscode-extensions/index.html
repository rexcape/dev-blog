我安装的 VSCode 插件

我安装的 VSCode 插件及主要功能

功能插件

插件 功能
Auto Comment Blocks 自动生成注释块
Better Comments 代码中的人性化注释，比如警告、质疑、高亮、TODO 等
Bracket Pair Colorizer 2 括号颜色高亮
C/C++ C/C++语法扩展
Code Time 统计写代码的时间
Comment Anchors 在评论或字符串中放置锚点,从而使导航大型文件更加容易。
Comment Divider 快速分隔注释
Data Preview 用表格、文本、图标方式预览 json 等数据文件
docs-markdown Markdown 写作帮助
ESLint ECMAScript 代码提示
Git Graph Git 可视化显示
GitLens Git 加强
jshint JavaScript 语言提示
Live Server 实时预览 Web 页面
Live Share 共享代码
Markdown All in One Markdown 写作工具
Markdown Preview Enhanced 更强力的 Markdown 预览
markdownlint Markdown 写作提示
npm Intellisense 可自动完成导入语句中的 npm 模块
Path Intellisense 路径自动补全
Polacode 将代码转换为图片
Prettier 前端代码美化工具
Remote - SSH SSH 远程开发
Remote - WSL WSL 开发
Rust Rust 语言工具
shellman Shell 脚本自动补全
Todo Tree 快速查看代码中的 TODO
Trailing Spaces 突出显示代码后面多余的空格
Web Template Studio 互联网应用框架脚手架，会捆绑安装 Azure 相关插件
