vw 和 vh

vw 和 vh 即是 viewport width 和 viewport height 的缩写，viewport 是目前看到的画面的大小，他们的值只能是 1 - 100，50vw 是占画面宽度的一半，50vh 是占画面高度的一半

经常用到的是 100vh 和 100 vw，用来做水平垂直居中都很方便
