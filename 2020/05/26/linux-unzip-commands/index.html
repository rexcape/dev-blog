Linux 常用解压命令

Linux 常用的解压命令

概述

Linux 下常见的压缩包格式为 zip tar.gz tar.bz2 tar.xz tar.Z
其中 tar 是打包格式，gz2 bz2 等后缀是指代压缩方式

filename.zip

filename.tar.gz

-z：gzip - 压缩格式
-x：extract - 解压
-v：verbose - 详细信息
-f：file - 文件

filename.tar.bz2

-j：bz2 压缩方式

filename.tar.xz

注意 J 大写

filename.tar.Z

注意 Z 大写

简便写法

首先，使用 tar --version 查看版本，从 1.15 版本以后可以直接使用 tar -xvf 来智能识别格式
