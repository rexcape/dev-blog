MySQL Yum Repo 安装链接

这个链接是在 MySQL Community Downloads - MySQL Yum Repository 下找到的，根据 Linux 版本来选取，比如说我安装的是 CentOS7 就选择 RedHat Enterprise Linux 7

点击之后出来的这个页面，复制下面的 no thanks 链接地址
