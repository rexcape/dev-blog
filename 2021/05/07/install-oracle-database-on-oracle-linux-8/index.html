由于我还是是初学者，不懂其中的原理，所以大多数的操作都是从网上搜索得来的。使用 RPM 安装的方法我重复了10次，最后还是无法进行连接。下面的步骤我重复了 8 次，最后成功使用 Navicat 软件连接。每个人环境都不相同，仅提供参考

环境

硬件环境

VMWare 虚拟机

软件环境

  • Oracle Linux 8.3
  • Oracle Database Enterprise Edition 19c
